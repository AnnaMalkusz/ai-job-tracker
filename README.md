# AI Job Application Tracker

> **Note:** This repository contains the **Portfolio Edition** of the application, prepared for demonstration and recruitment purposes. It showcases the complete recruitment workflow and key features while using fictional data to protect personal information.

*A modern AI-assisted recruitment workspace that helps organize job applications, interview preparation and recruitment workflows in one structured system.*

# ✨ Features

- AI-assisted recruitment workspace
- Job application management
- Smart search & filtering
- Structured interview preparation
- Recruitment notes & contact history
- Salary tracking
- Quick Import workflow
- PDF export
- Local-first architecture
